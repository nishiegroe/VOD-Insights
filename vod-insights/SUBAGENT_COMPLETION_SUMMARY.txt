BACKEND REQUIREMENTS
================================================================================

To fully enable this feature, backend must provide:

1. GET /api/vods/list
   Returns: { "vods": [{ path, name, display_name, duration, resolution, ... }] }

2. POST /api/sessions/multi-vod
   Accepts: { vod_paths: [...], session_name: string, sync_mode: string }
   Returns: { ok: true, session_id: string } or { ok: false, error: string }

See FRONTEND_IMPLEMENTATION.md for full API specifications.
